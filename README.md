# Outlook VBA 邮件附件批量提取工具
Lightweight Outlook VBA Tool for Batch Email Attachment Extraction

## 项目说明
本项目完工于 **2024年12月**
目前可能已存在更高效的邮件附件导出方案，本仓库仅用于留存本人**稳定实现**的批量提取功能版本。

## 运行要求
⚠️ 必须使用 **Outlook (classic)** 运行
官方下载链接：https://go.microsoft.com/fwlink/?linkid=2276901

## 核心功能
批量提取邮件附件，支持筛选与自动重命名，无第三方依赖，适用于办公自动化。
